# i茅台 使用教程


注意：仅提供专业版系统才可安装！非专业版请勿安装，否则安装过程失败，将需要执行purge软件包的操作。  


## 安装与卸载

- 安装

```shell
apt install imaotai-histb -y
```
- 卸载

```shell
apt remove imaotai-histb
```

安装完成后访问 `http://IP:9999` 即可
