# 海纳思系统 使用手册介绍

本手册由作者神雕热心撰写。由于工作量大，未详尽之处，敬请理解。

## 海纳思系统介绍：

英文名称 Hi-NAS ，中文名称：海纳思 

以海思机顶盒为开发平台，打造出一个简单易用的小型家用级服务器系统，

又有海纳百川之意。

## 初衷

为了让国内用户向普及 Linux 系统方向不断发展和进步，

把嵌入式的 NAS 系统尽可能的以浅显的描述方式展现出来，

让使用者从小白到摸进门槛，再到自主创作个人需求的使用方式迈进。
